A comprehensive collection of assignments, algorithms, and projects completed during the Autumn 2025 native iOS Development course at **Kazakh-British Technical University (KBTU)**.

## 🎯 Overview
This repository reflects my progression in iOS development, covering everything from fundamental Swift syntax and Object-Oriented Programming (OOP) to UI rendering and data structures.

## 🧠 Key Topics Covered
* **Swift Fundamentals:** Optionals, Closures, Protocols, Generics, and Memory Management (ARC).
* **Architecture:** MVC (Model-View-Controller) implementation.
* **UI Development:** Building interfaces programmatically and via Storyboards (UIKit).
* **Data Flow:** Working with Collections, Data Models, and UserDefaults.
